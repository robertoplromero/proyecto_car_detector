Proyecto de detector de autos
Materia: DeepLearning
Profesora: Dra. Lorena Díaz González

El proyecto aborda el uso de modelos basados en YOLO con un amplio DATASET que por defecto está segmentado.

En este trabajo se genera un split con un procesamiento adicional para comprobar la confiabilidad

Se realizan 3 modelos
* El primero es el split original
* El segundo es un resplit
* El tercero es un resplit desde 0 (sin pesos)


Se utiliza YOLOv8n para los tres casos
